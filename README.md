
## 概述：
```
此框架组件库是为了日常积累常用基本平台组件，UI组件及工具类目的而创建。
```

### 组件环境配置与安装
- 1. 安装node，npm版本3以上。
- 2. 设置npm组件库
npm config set registry http://registry.npmjs.org/

- 3.使用next.js,ant design pro,react-create-app等React 框架建立项目
- 4.项目目录下安装：
   npm install -D simple-framework
